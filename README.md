# AthexClient
AthexClient is a open source client to use for Agar.io private servers. Feel free to use this client to play with your friends or make a game, but please give credit :)

# Features
  - Minimap
  - Chat
  - Team Tag
  - Leaderboard Color
  - Custom Hats/Crowns
  - A lot of Settings
  - A lot of Theming options
  - Hotkeys
  - Server Panel
  - Much more!
